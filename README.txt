************************************************************************************
                PACKAGES AND SOFTWARES NEEDED
************************************************************************************

gcc compiler - for executing C PROGRAM
python3      - for executiong python programs

************************************************************************************
                        USAGE
************************************************************************************

Step 1: Unzip the group9.zip folder and copy the input trace files into the
        extracted folder.

Step 2: Execute read_trace.c to extract trace files using
            1. g++ read_trace.c -o read_trace
            2. ./read_trace

        This step generates .txt files which will be used by python programs

Step 3: Execute python programs to execute Question 1 and 2 using
            1. python3 inclusive.py
                    - Simulates inclusive L2 and L3 caches and computes L2, L3
                      cache misses.
            2. python3 exclusive_simulator.py
                    - Simulates exclusive L2 and L3 caches and computes L2, L3
                      cache misses.
            3. python3 nine_simulator.py
                    - Simulates NINE L2 and L3 caches and computes L2, L3
                      cache misses.
            4. python3 Ques2.py
                    - Simulates fully associative L2 and L3 caches and computes
                      L2, L3 cache misses.
